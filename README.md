# توفيق أسست (Tawfeeq Assist) - مشاركة الشاشة للدعم الفني

أداة سحابية مستقلة وخاصة تتيح لمهندسي الدعم الفني بث ومشاركة شاشة العميل بالوقت الحقيقي عبر متصفح الويب مباشرة، باستخدام تقنيات **WebRTC** و **WebSockets** للاتصال المشفر والمباشر (نظير لنظير P2P)، بدون اشتراكات وبخصوصية تامة.

---

## الميزات الرئيسية
* **اتصال P2P مباشر:** يتم نقل البث مباشرة بين متصفح المهندس ومتصفح العميل لحماية الخصوصية وزيادة السرعة.
* **بدون تثبيت برامج للعميل:** يكفي أن يفتح العميل الرابط من متصفحه على الكمبيوتر أو الأندرويد ويبدأ البث.
* **لوحة تحكم للمهندس:** تدعم وضع ملء الشاشة، تكبير البث (Zoom)، والتقاط لقطات شاشة وحفظها تلقائياً بصيغة PNG.
* **استضافة ذاتية كاملة:** لا تعتمد على أي خوادم طرف ثالث، والتحكم بالكامل لشركتك.

---

## التشغيل المحلي (Localhost)

1. تأكد من تثبيت **Node.js** على جهازك.
2. افتح منفذ الأوامر في مجلد المشروع وقم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```
3. قم بتشغيل خادم الإشارات:
   ```bash
   npm start
   ```
4. افتح الرابط التالي في متصفحك للتجربة:
   [http://localhost:3000](http://localhost:3000)

---

## كيفية الرفع والتشغيل المجاني على Render.com

1. قم بإنشاء مستودع (Repository) جديد على حسابك في **GitHub** (يمكن أن يكون خاصاً Private أو عاماً Public).
2. ارفع ملفات هذا المشروع إلى المستودع (تأكد من عدم رفع مجلد `node_modules` حيث تم استبعاده تلقائياً عبر ملف `.gitignore`).
3. توجه إلى موقع [Render.com](https://render.com) وسجل دخولك مجاناً.
4. اضغط على زر **New +** ثم اختر **Web Service**.
5. اربط حساب GitHub الخاص بك، ثم اختر المستودع الخاص بالأداة.
6. سيقوم موقع Render بالتعرف تلقائياً على لغة المشروع كـ **Node**.
7. تأكد من الإعدادات التالية:
   * **Build Command:** `npm install`
   * **Start Command:** `npm start`
8. اضغط على **Deploy Web Service**.
9. بمجرد اكتمال الرفع، سيعطيك الموقع رابطاً آمناً يبدأ بـ **`https://`** (مثال: `https://your-app-name.onrender.com`).
10. افتح الرابط وابدأ بتجربة الأداة ومشاركة الشاشة بنجاح مع أي شخص عبر الإنترنت!
